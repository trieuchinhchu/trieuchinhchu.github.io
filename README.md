# CHU TRIEU CHINH – Senior Backend Engineer

Interactive personal resume website built with HTML, Tailwind CSS, and vanilla JavaScript.

**Live:** [trieuchinhchu.github.io](https://trieuchinhchu.github.io)

## Features
- Multi-page project details with metrics & badges
- Dark mode & multi-language (EN/VI)
- Responsive design
- Animated counters & scroll effects
- PDF download via browser print
